# ANSI Drawing

Making drawings using ANSI chars. Primitives are labels (text), frames (line
around area), rectangle (filled area), lines (braille points) etc.

## Platform

- Shell: Windows 11 Powershell
- Programming language: TypeScript
- Check type command: deno check
- Linting: deno lint
- Testing: deno test

## Dependencies

Using CharPlot library from jsr:@sauber/widgets for plotting characters one at a
time onto a char canvas.

## Development Process

- Test driven development, write test cases before code
- Validate and fix one test case at a time, before proceeding to next
- SOLID principles for organizing code
